Главное меню

Главное окно программы представляет из себя двойное меню, расширенное и упрощенное.
Упрощенное меню. Упрощенное меню имеет основные элементы управления:
  • Файл
    1. Новый
    2. Открыть
    3. Сохранить
  • Эмулятор
    1. Эмуляция
    2. Отладка
    3. Компилировать в EXE

Рис.1-Главное окно программы
Теперь вкрадце о каждом пункте основного меню:
Меню "Файл", в данном меню содержатся пункты:
  1. Создать
  2. Открыть
  3. Сохранить
  4. Выход

Меню "Создать" позволяет создать новый проект или создать проект по уже готовому шаблону.
Меню "Открыть" позволяет открыть уже существующий проект.
Меню "Сохранить" записывает текущий проект в файл.
Меню "Выход" завершает работу с программой.

Меню "Правка", в данном пункте меню содержатся пункты:
  1. Копировать
  2. Вырезать
  3. Вставить
  4. Удалить
  5. Вставить кусок кода

Меню "Копировать" позволяет скопировать выделенный в редакторе текст в буффер обмена (с удалением старого содержимого).
Меню "Вырезать" удалаят собержимое буффера обиена, помещает выделенный текст в буффер обмена и удалает его из редактора.
Меню "Вставить" позволяет вставить содержимое буффера обмена в редактор кода.
Меню "Удалить" удаляет выделенный текст в редакторе кода.

Так же все эти команды меню присутствуют внути редактора кода как контекстное меню.

Меню переменных
  1. Управление переменными
  2. Очистить список переменных
"Управление переменными", данная команда позволяет управлять всеми глобальными переменными в проекте, можно добавлять, удалять, изменять переменные и их значения.
"Очистка списка переменных" применяется в том случае, если обрабатываемая программа критична к переменным, т.е берет значение непосредственно из списка.

Конфигурации
  1. Настройки проекта
  2. Подсветка синтаксиса
"Настройки проекта". Эта команда вызывает окно глобальных настроек проекта, таких как имя файла, его расширение, иконка. Так же предусмотрена запаковка выходного файла с помощью запаковщика PE файлов-UPX. Для защиты программы так же есть возможность шифрования выходного файла, для этого необходимо установить установить флажок "Set psaaword" и в появившемся поле ввода ввести пароль.
"Подсветка синтаксиса" позволяет подсветить ключевые слова обрабатываемого кода.

Program. Меню работы с текущим кодом, содержит два элемента.
  1. Emulate
  2. Compile to EXE
Emulate-эмуляция текущей программы без остановок на отладку (возможны прерывания на ввод значений, если используется консоль).
Compile to EXE-преобразует текущий проект в *.EXE файл, так же задает некоторые параметры заданные в опциях проекта.

Debug-меню для отладки текущей программы.
  1. Debug run
  2. Step over
  3. Tracing into
  4. Reset
Debug run-запуск текущей программы пошагово в отладчике.
Step over-делает шаг вперед, не прерываясь на вложенные функции.
Tracing into-Делает шаг вперед, прерываясь на вложенные функции.
Reset-останов отладчика и выполнения текущей программы.
Windows-меню быстрого доступа к основным окнам программы
  1. Variable list
  2. Code window
  3. 2D view
  4. 3D view
  5. Console
Help-пункт меню отображающий справку и информацию о программе.
