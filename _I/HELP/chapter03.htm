Редактор кода

Редактор кода-это одно из основных окон, необходимых для обработки программного кода. Окно представляет из себя простейший текстоывй редактор с функциями форматирования текста. Так же в редакторе предусмотрена возможность подсветки синтаксиса для более удобного чтения и быстрой навигации по программному коду.


Рис.1 - Внешний вид окна редактора кода.

Редактор состоит из трех частей:
  1. Табулятор вкладок
  2. Редактор
  3. Окно статуса
Табулятор вкладок предназначен для быстрого переключения между несколькими проектами.
Редактор-простой текстовый редактор с подсветкой синтаксиса, позволяет копировать, вырезать, вставлять текст, а так же осуществлять поиск по тексту.
Окно статуса предназначено для отображения хода работы программы, вывода сообщений об ошибках.

Поиск текста
В программе предусмотен быстрый поиск текста. Для вызова поиска текста используется сочетание горячих клавиш Ctrl+F.


Рис.6 - Окно поиска текста.

В данном окне необходимо указать искомое слово, поционально указывается регистр букв, после чего программа попытается найти искомый текст. Для продолжения поиска достаточно нажать кнопку "Найти далее".
