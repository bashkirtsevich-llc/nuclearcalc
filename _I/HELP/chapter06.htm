Отладчик

Отладчик представляет собой отдельный модуль программы, проедназначенный для проверки работоспособности программы и исправления ошибок допущенных при разработке программы.
Основные элементы управления отладчиком:
  • Trace into
  • Step over
  • Halt
  • Continue

"Trace into"-в данном режиме происходит отладка строки кода с пошаговой отладкой всего кода на данной строке
"Step over"-На данном режиме отладки происходит обработка кода на данной строке, иными словами отладчик автоматически обрабатывает строку кода на данной строке.
"Halt"-Завершение обработки кода и выход из режима отладки. В данном случае произойдет останов интерпретатора кода.
"Continue"-Данная команда сообщит интерпретатору об отмене отладки и продолжению основного кода без остановочно.


Дополнительные элементы отладчика:
  • Окно с кодом
  • Список обработанных функций
  • Список переменных
  • Отчет отладки

Окно с кодом-в этом окне отображается основной код отлаживаемой программы, со всем содержимым подключаемых модулей.
Список обрабатываемых функций-в этом списке отображаетсясама обрабатываемая функция и ее результат.
Список переменных-в этом списке отображаются и обновляются все глобальные переменные и их значения.
Отчет отладки-в этом списке отображаются все имена обрабатываемых процедур, код внутри цикла, т.е вся программа представляется в виде последовательности команд без переходов и циклов.
