3D Режим

В программе предусмотрена работа в трех мернойм режиме, используя технологию отрисовки OpenGL.


Рис.5 - Внешний вид окна OpenGL

Для работы в режиме OpenGL, неоюходимо запросить интрпретатор на включение режима, для этого используется команда Init3D(_CREATE); внутренний параметр функции переключает режим работы (активировать/деактивировать).
Для построения точки в пространстве есть функция добавления вершины glVertex3f(x,y,z);, параметром данной функции является координата точки в плоскаостях ZX,YX,YZ.
Для построения вектора в пространстве необходимо указывать 2 точки, предварительно установив параметр отрисовки glBegin(_GL_LINES);
Для завершения этапа отрисовки используется команда glEnd(void).
Дополнительные параметры:
  1. _GL_POINTS-запускает отрисовку графика точками
  2. _GL_LINES-запускае отрисовку графика векторами, необходимо указывать 2 вершины
