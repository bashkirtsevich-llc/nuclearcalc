Окно переменных

Данное окно предназначено для управления глобальными переменными.


Рис.3 - Окно переменных.

Окно имеет довольно простой интерфейс, таблица переменных, кнопки управления переменными:
  1. Добавить
  2. Удалить
  3. Очистить
Добавить новую переменную, необхдимо ввести имя переменной и ее значение, если переменная строкового типа, то значение необходимо взять в двойные кавычки.
Удалить выбранную переменную, если есть необходимость удаления переменной.
Очистить весь список переменных, эта операция может понадобиться при перезапуске текущей программы.

Добавление переменных
Для добавления переменной необходимо перейти в окно редактирования переменных, после чего нажать кнопку "Добавить", при нажатии на кнопку появится модальное окно.


Рис.4 - Окно добавления переменных.

По завершению ввода нужной информации и нажатии кнопки "Ок", переменная и ее значение тут же добавятся в список и сразу же буду тдоступны в обрабатываемой программе.

Окно структуры проекта

Рис.5 - Окно структуры проекта.

Данное окно отображает общую структуру текущего проекта.
В данном окне содержится информация о имени проекта, файле с основным кодом проекта, подключенных модулях, иконке проекта.
Содержимое окна изменяется при переключении между проектами.
