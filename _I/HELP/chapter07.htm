Настройка параметров

Для управления проектом использется меню настроек, вызываемое комбиныцией клавиш Ctrl+Shift+F11, в появившемся окне можно изменить параметры текущего проекта, такие как:
  1. Подключаемые модули
  2. Заголовок/Иконка/Расширение компилируемого файла
  3. Защита исполняемого файла

Рис.3-Окно настроек программы
Подключаемые модули можно использовать в тех случаях, когда проект является очень большим, можно разнести разные части одной программы в модули, хранящиеся в папке "MODULES". Для использования модуля в программе достаточно поставить галочку напротив имени файла-модуля, после чего его можно сразу же задействовать в текущем проекте.
Заголовок/Иконка/Расширение компилируемого файла, настраиваются в том случае, если есть необходимось скомпилировать текущий проект в исполняемый файл. В этом разделе указывается имя проекта, его иконка (по выбору), исходное имя (по умолчанию .exe)
Защита исполняемого файла используется опционально, если есть смысл конфиденциальности. В этом пункте настроек достаточно установить галочку и ввести пароль.
