*Работа в двухмерном режиме*

Для работы в двухмерном режиме необходимо перевести программу в режим графопостроителя, "Главное меню->Режим->2D/3D графопостроитель". В появившемся окне, выбрать пункт меню "Файл->Новый->2D График".
Далее появится окно, в котором будет необходимо указать функцию, которую необходимо чертить, так же указать параметры, такие как цыет графика, начальное, конечное значение предела и количество шагов изменения значения.
По центру окна будет отображен график, а так же для удобства будет начерчены координатные оси, координатрая сетка(опционально отключаются).
Программа так же позволяет чертить подряд сразу несколько графиков, это может быть удобно для оптического анализирования.
Для увеличения/уменьшения графика, используется колесо мыши.
