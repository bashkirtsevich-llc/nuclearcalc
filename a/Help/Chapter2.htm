[Atomic Calculator]



*Выполнение расчетов с применением функций*

В составе программы имеется большой набор функций, среди них:
  • Тригонометрические
  • Гиперболические
  • Математические
  • Числовые
  • Функции работы с текстом
  • Функции работы с множествами
  • Логические функции

Для получения значения какой-либо функции необходимо ввести имя функции, в скобках казать аргумент функции, если это тригонометрические функции, то значение указывается в радианах.
В качестве примера можно ввести следующий текст Sin(1); после нажатия кнопки выполнения, программа выведет результат.В качестве параметра может быть задана какая либо переменная.
Так же можно присвоить результат функции какой либо переменной.
